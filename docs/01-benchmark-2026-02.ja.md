## 3つの意外な発見

### 1. 🤯 GPT-4o-miniは基本的な計算すらできない
「17 + 35」を聞いたところ → **54** と回答（完全な問題の正解：48）。推論スコア：30/100。
AI Agentの計算処理をGPT-4o-miniに頼っているなら、問題です。

### 2. 💪 無料の8Bモデルがを上回る
Cerebras llama3.1-8b（無料・80億パラメータ）のスコアは **92**。GPT-4o-miniは **82**。
レイテンシ316ms。無料。GPTより優秀。

### 3. ⚡ 最速 ≠ 最良
Groqは平均の8倍速（306ms）ですが、中国語スコアは **30/100** まで低下。
非英語Agentにとって、多言語品質を伴わない速度は罠でしかありません。

## ⚠️ APIフィールド名の罠 5選

これらのフィールド名は想定と**異なります**。間違えるとサイレントエラーの原因に：

| API | ❌ 想定 | ✅ 実際 |
|-----|---------|---------|
| Vision | `imageUrl` | `image` |
| Geocode | `query` | `q` |
| CoinGecko | `coin` | `coins` |
| Serper | `results` | `organic` |
| X Search | `results` | `tweets` |

## テスト手法

1. **実際のAPIコール** — 合成ベンチマークではありません。全ての数値は実際のHTTPリクエストによるものです。
2. **各APIにつき4回テスト** — 各テストは4回実行し、ばらつきを考慮しています。
3. **東京から実施** — 全テストは東京サーバー（AWS ap-northeast-1）から実行。
4. **オープンな採点基準** — 推論＝数学の正確性、コード＝関数出力、多言語＝CN/JP/ENの精度。
5. **スポンサーなし** — ランキングは純粋にデータに基づいています。全API費用は自己負担です。
